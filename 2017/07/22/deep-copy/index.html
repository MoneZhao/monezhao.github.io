利用序列化实现对象的拷贝

发表于 2017-07-22 | 更新于 2018-03-15
本文字数： 2.3k | 阅读时长 ≈ 4 分钟

如何利用序列化来完成对象的拷贝呢？在内存中通过字节流的拷贝是比较容易实现的。把母对象写入到一个字节流中, 再从字节流中将其读出来, 这样就可以创建一个新的对象了, 并且该新对象与母对象之间并不存在引用共享的问题, 真正实现对象的深拷贝。

使用该工具类的对象必须要实现 Serializable 接口, 否则是没有办法实现克隆的。
