整个流程

  1. 客户端 http 请求服务器 80 端口，该端口被映射到 Nginx 容器 80 端口，进入 Nginx 处理。
  2. Nginx 分析请求，如果是静态资源，直接服务器读取内容；如果是 PHP 脚本，通过 PHP 容器调用服务器获取脚本，然后 FastCGI 处理。
  3. FastCGI 解析 PHP 脚本，必要时访问 MySQL 容器读写数据。
